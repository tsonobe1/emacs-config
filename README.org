* My Emacs 設定

このリポジトリは [[file:config/myinit.org][config/myinit.org]] を本体として
[[file:config/myinit.el][config/myinit.el]] を org-babel tangle で生成します。

設定の再生成コマンド:

#+begin_src sh
emacs --batch -Q -l scripts/tangle-myinit.el
#+end_src

smoke テスト実行コマンド:

#+begin_src sh
emacs --batch -Q -l test/config-smoke.el -f ert-run-tests-batch-and-exit
#+end_src
