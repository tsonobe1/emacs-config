* My Emacs config

This repository is maintained from [[file:config/myinit.org][config/myinit.org]] and generated to
[[file:config/myinit.el][config/myinit.el]] by org-babel tangle.

Regenerate the tangled config artifact with:

#+begin_src sh
emacs --batch -Q -l scripts/tangle-myinit.el
#+end_src

Run smoke tests:

#+begin_src sh
emacs --batch -Q -l test/config-smoke.el -f ert-run-tests-batch-and-exit
#+end_src
